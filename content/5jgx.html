網誌與簡報:

利用 cmsimde 建立的網際內容管理, 除了 Web site 外, 還提供 blog 與 reveal 等兩項功能.

blog 是網誌, 採用 Pelican 建置, 使用者可以利用  Leo Editor 編輯 config 目錄中的 pelican.leo, 以 Markdown 建立網誌原始資料後, 再利用  Pelican 將原始網誌檔案轉為超文件格式的網誌內容, 轉出的網誌檔案將存入 blog 目錄中.

而 reveal 則是網際簡報的功能, 使用 reveal.js 建立, 使用者可以利用 Leo Editor 編輯 config 目錄中的 reveal.leo, 以 html 或 Markdown 格式建立簡報, 網際簡報檔案則位於 reveal 目錄中.

自建網際伺服器:

cad2 與 cad5 伺服器:

cad2 Ubuntu 22.04 server

cad5 Ubuntu 22.04 desktop

將分別建立對應的使用者帳號, 以便用來配置各分組的動態與靜態網站.
